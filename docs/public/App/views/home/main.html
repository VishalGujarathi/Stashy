Stashy

A collection of components for doing responsive things

  Download   View project on Github

What is this ?

Stashy is a library trying to help with responsive designs. Inside, you can find layouts and utilities perhaps could be useful for you.

Components in this library are mainly focused in layouts, so you have to style them properly on your own for look and feel like you are looking for. For example, this site is using the awesome Bootstrap plus some personal css.

All pages in this site were built using Stashy, so you can check what can you do with this components going around the docs.

Components inside this library

OffCanvas

Offcanvas layouts have the ability to adapt to any screen size by hiding / showing panels when reaching certain break points.

This page, and most in these docs are offcanvas layouts, so resize your browser to see them in action. In small sizes, click the     buttons for showing side panels.

Go to OffCanvas docs

Flyout

Creates a left menu showing and hiding its content on user interactions.

You can see this component in web and mobile applications like Facebook or Google ones.

Go to Flyout docs

Toogle Menu

Top menu showing all its links in large screens and hiding them on smaller ones for showing a button to slide the menu down.

Menu at the top of this page is a sample of Toggle using Stashy. Resize your browser window to see it in action.

Go to Toggle docs

ShowMeMore

This utility allow hiding list items and showing a "Show more" button.

Go to ShowMeMore docs

Focal Point

Cropping images at client side inspired in the focal point technique.

Go to FocalPoint docs

Loader

Nice loader for your responsive designs inspired in the Windows 8 / Windows Phone one.

Go to Loader docs

Standalone samples

You can check each component across the docs, but it´s easy to get started by simply checking the standalone samples in the Github Project.

Each sample is build in a individual page and only the minimal dependencies and markup is coded.

Check it!!

Dependencies

The entire library has a dependency with jQuery so it is necessary to include it before any of the Stashy javascript files.

Perhaps you´ll find another dependencies specific to some of the components included. You can check them in the specific component docs page.
