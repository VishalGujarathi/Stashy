Notify

Notications using toast, bars and panels styles

  Download   View project on Github

Description

Most applications needs to communicate info to the end user. With Notify you can do it in a unobstrusive and responsive way.

There are three types of notifications included:

toast Opened in the corner you choose and 40% width.

bar Opened top or bottom and full width.

panel Opened left or right and full height.

Javascript

Constructor

Stashy.Notify(userOptions)
Parameter Type Description
userOptions object The initialization options for each instance. Take a look to the following table and explore the available options

The userOptions object

Option Type Default Description
target string body The element target container
title string empty string The element title
titleSize int 3 You can enter wathever value in the 1..6 range.The title will be a HTML heading. This number will provide the heading size (h1, h2, h3, ...)
contentType string text

Can be "text" or "html".

If you select text, then the content property should be simple text.

On the other side, if you select "html", you can provide inline html or a CSS selector to choose what to include in the notification.

If you option is "html" and providing a selector remember to add the st-notify-content class to the content you are going to include in the notification in order to hide it.
content string empty string You can add simple text, inline html or a CSS selector
style string default It change the color of the notification. The allowed styles are "default", "error", "info", "success" and "warning"
animDuration string fast "fast" or "slow" for a 1 sec or 2 sec animation
closeArea string button

If you select "button" then a button will be added to close the panel.

If you choose "element" then the element will be clickable for closing it.

Methods

Name Description
toast(positionX, positionY,radius Create a toast style notification

positionX: Can choose "right" or "left"

positionY: "top" or "bottom"

radius: Select true if you like rounded corners

bar(positionY) Creates a bar style notification

positionY: "top" or "bottom"

panel(positionX) Creates a panel style notification

positionX: Can choose "right" or "left"

Sample

Stashy.Notify({
    title : "My toast",
    content : "Toast style notification",
    titleSize : 4,
    style : "success",
    contentType : "text",
    animDuration : "fast",
    closeArea : "element"
}).toast("right", "bottom", true);

Stashy.Notify({
    title : "My bar",
    content : "Bar style notification",
    titleSize : 1,
    style : "info",
    contentType : "html",
    animDuration : "slow",
    closeArea : "button"
}).bar("top");

Stashy.Notify({
    title : "My panel",
    content : "#wathever-element-in-the-page",
    titleSize : 2,
    style : "default",
    contentType : "html",
    animDuration : "fast",
    closeArea : "button" 
}).panel("right");
